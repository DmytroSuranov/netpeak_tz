На домашней странице проекта нужно выбрать интересующую социальную сеть.

1. Для работы с Twitter'ом достаточно ввести имя учетной записи и нажать на кнопку "Загрузить ленту" (информация обновляется каждый 10 секунд).

2. Для вывода ленты с Facebook'a нужно авторизироваться под учетной записью, ленту который нужно просмотреть (информация обновляется каждый 10 секунд).
В связи с политикой конфиденциальности простое API не имеет доступа к постам пользователя. Для получения уровня доступа нужно отправить заявку в фейсбук и получить их одобрение на использование данных для API.
Но есть возможность добавить пользователей в качестве тестеровщиков API. Для них все данные доступны при использовании приложения.
